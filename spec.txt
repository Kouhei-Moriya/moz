[misc]
   inst  |argument|description
*nop     |        | nop

[control flow]
   inst  |argument|description
*succ    |        | always succed
*fail    |        | always fail
*jump    | L      | jump to the label L always
*jumpif  | L      | jump to the label L if the result is F
*call    | L      | call the nonterinal expression that is labeled L
*ret     |        | return from the nonterinal expression

[matcher]
   inst  |argument|description
*char    | C      | match the character 'C'
*any     |        | match the any character
*class   | M      | match tha character class (M=[a-z])

[stack operation]
   inst  |argument|description
*push    |        | push the current position to the vm stack
*pop     |        | pop  the current position from the vm stack
